# unloan PWA 🚀

AI-powered financial freedom planner to help you reduce loan dependence.

## Features
- ✅ Personalized financial health score
- ✅ Insurance gap analysis
- ✅ Milestone goal tracking
- ✅ Short & long-term investment recommendations
- ✅ Google Sheets cloud storage
- ✅ Works offline (PWA)
